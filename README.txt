Product Calculator

Готовый просмотр без Node.js / IDE:
1. Откройте папку ProductCalculator.
2. Откройте файл dist\index.html в браузере.
3. Этот файл самодостаточный: стили и JavaScript уже встроены внутрь него.

Исходный код приложения:
- src\App.jsx
- src\components\ProductPanel.jsx
- src\App.css
- src\index.css

Запуск для разработки, если установлен Node.js:
1. Откройте терминал в папке ProductCalculator.
2. Выполните npm install.
3. Выполните npm run dev.
4. Откройте адрес, который покажет Vite.

Production-сборка, если установлен Node.js:
1. Откройте терминал в папке ProductCalculator.
2. Выполните npm install.
3. Выполните npm run build.
4. Готовые файлы появятся в папке dist.
5. Для прямого открытия через file:// используйте уже приложенный dist\index.html.

Примечание:
Папка node_modules не нужна для проверки исходного кода и не обязательна для просмотра готовой версии из dist.
