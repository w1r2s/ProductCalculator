# Product Calculator

React-приложение для ввода списка продуктов и расчета общей суммы.

## Возможности

- Добавление и удаление строк продуктов.
- Защита от удаления последней строки.
- Автофокус на поле "Наименование" после добавления новой строки.
- Валидация количества и цены:
  - количество: положительное число, до 5 знаков после запятой;
  - цена: положительное число, до 2 знаков после запятой.
- Расчет итоговой суммы с округлением до двух знаков.
- Прокрутка списка при добавлении седьмой панели.

## Запуск

```bash
npm install
npm run dev
```

## Production-сборка

```bash
npm run build
npm run preview
```

## Просмотр без Node.js

В архивной версии проекта есть готовый файл:

```text
dist/index.html
```

Его можно открыть напрямую в браузере.
